# WeatherAppNational
기상청 API를 이용한 날씨앱

* 사용 예정 기능
  * 기상청 API parse&fetch
  * UIPageViewController를 사용한 화면 넘김 및 화면 추가
  * 현재위치사용 및 기상청 API용 격자 변환
  * firebase를 통한 로그인/로그아웃
  * firebase를 통한 게시판 기능
